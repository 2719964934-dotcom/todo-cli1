# Todo CLI - 命令行待办事项工具

一个简单但完整的命令行待办事项管理工具，支持添加、查看、完成、删除任务，数据自动保存到 JSON 文件。

## 功能特性

- ✅ 添加任务：`add 任务内容`
- 📋 查看所有任务：`list`（显示序号、状态、内容）
- ✔️ 标记完成：`done 序号`
- ❌ 删除任务：`delete 序号`
- 💾 自动保存：退出时自动保存到 `todos.json`
- 🌏 支持中文（UTF-8 编码）

## 安装与运行

### 环境要求
- Python 3.6 及以上（无需额外安装第三方库，仅用标准库）
